# Getting help

- **Look it up first.** Open the repo as an Obsidian vault and start at
  [`wiki/Rust Brain Home.md`](wiki/Rust%20Brain%20Home.md), or browse the full
  catalogue in [`wiki/index.md`](wiki/index.md). The cross-cutting "do this,
  avoid that" summary is in
  [`research/99-rust-mastery-synthesis.md`](research/99-rust-mastery-synthesis.md).
- **Ask a question.** Open an issue using the **Note correction** or
  **Coverage request** [templates](../../issues/new/choose). For broader Rust
  and AI-tooling conversation, this project is part of the
  [AI Marketing Hub Pro](https://www.skool.com/ai-marketing-hub-pro) community.
- **Contribute a fix.** See [`CONTRIBUTING.md`](CONTRIBUTING.md).
- **Report something sensitive.** See [`SECURITY.md`](SECURITY.md).

This is a community knowledge base, not a support desk, so responses are
best-effort.
