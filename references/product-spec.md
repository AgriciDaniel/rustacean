# Product Spec — Rust Brain

**Buyer / user:** Rust developers who want to write correct, idiomatic, current (edition 2024) Rust with confidence.

**Outputs (deliverables):**
- Cross-linked concept notes for the Rust language and standard library
- Idiom / best-practice pattern notes
- Anti-pattern / footgun notes with the correct alternative
- Domain maps (MOCs) and a mastery synthesis

**Refresh cadence:** every 6-week Rust release for language/std; monthly for ecosystem & security.
**Boundaries:** read-only knowledge brain; cites official docs/RFCs/source; no code execution against user systems.
See `specs/rust-brain.yaml`, `references/source-ledger.json`, `references/safety-gates.md`.
